We are going to build an agent who is going to play and learn using penalties and rewards that it gets from each stage. There are 4 locations which are labeled by different letters: R, G, B and Y. Our goal is to pick up the passenger from one location and drop him off at another. This is a text based environment. So everything is in ASCII character.
